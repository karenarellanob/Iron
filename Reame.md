# Iron - Minimalist Weight Tracking 🏋️‍♀️

**Iron** es una aplicación nativa para iOS diseñada para levantadores de pesas que valoran la velocidad y la claridad. Construida enteramente con **SwiftUI** y **SwiftData**, permite registrar entrenamientos sin fricciones, enfocándose en el progreso personal (PR) y la estética minimalista.

## 🔒 Política de Privacidad (Privacy Policy)

Tu privacidad es lo más importante. **Iron** sigue una filosofía de "Privacidad por Diseño":

1.  **Almacenamiento Local:** Todos tus datos de entrenamiento, ejercicios y récords personales se almacenan de forma local en tu dispositivo mediante **SwiftData**.
2.  **Sin Recolección de Datos:** No recolectamos, vendemos ni compartimos tu información personal con terceros.
3.  **Sin Registro Obligatorio:** No necesitas crear una cuenta para usar las funciones básicas de la app.
4.  **Compras In-App:** Las transacciones se gestionan de forma segura a través de **Apple StoreKit 2**. No tenemos acceso a tus datos de pago.

*Última actualización: 9 de abril de 2026.*

---

## 🚀 Tecnologías Utilizadas

* **Swift 6 / SwiftUI:** Interfaz declarativa y moderna.
* **SwiftData:** Gestión persistente de la base de datos local.
* **StoreKit 2:** Implementación segura de compras in-app (Plan Pro).
* **Charts Framework:** Visualización de datos y progreso.

## 🛠 Instalación para Revisores (App Store Review)

Para probar las funcionalidades de la aplicación durante la revisión:
1. Navegar a la pestaña **Ejercicios** para añadir un nuevo movimiento.
2. Registrar una serie en **Peso Muerto** para activar la lógica de PR (Personal Record).
3. Acceder a la pestaña **Pro** para simular la actualización mediante el entorno Sandbox.
